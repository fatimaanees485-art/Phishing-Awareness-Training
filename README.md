# Phishing Awareness Training

This folder contains a self-contained online training module about phishing attacks, fake websites, social engineering tactics, and practical prevention habits.

## File Included

- `phishing-awareness-training.html` - the full interactive training module.

## How To Open

Open `phishing-awareness-training.html` in any modern web browser.

If a local preview server is running, you can also open:

`http://localhost:8765/phishing-awareness-training.html`

## What The Module Covers

- How to recognize phishing emails.
- How to identify suspicious sender addresses, links, attachments, and urgent requests.
- How to spot fake websites and misleading URLs.
- Common social engineering tactics, including authority, urgency, secrecy, trust chaining, and reward-based lures.
- Best practices to avoid phishing, including verification, password managers, multi-factor authentication, and reporting.
- Real-world attack patterns such as account hold scams, business email compromise, and fake government portals.
- An interactive 5-question quiz with instant feedback and scoring.

## Suggested Use

This module can be used for:

- Employee cybersecurity awareness training.
- Classroom or workshop instruction.
- Self-paced online learning.
- A quick phishing refresher before audits, simulations, or security campaigns.

## Learner Takeaways

By the end of the module, learners should be able to:

- Pause before clicking links or opening unexpected attachments.
- Inspect sender addresses and website domains more carefully.
- Recognize emotional pressure tactics used by attackers.
- Verify sensitive requests through trusted channels.
- Report suspicious messages instead of engaging with them.
- Respond quickly if credentials are entered on a suspicious site.
